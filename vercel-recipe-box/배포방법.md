# Yumi G's Recipe — Vercel 배포 방법

코딩을 몰라도 아래 순서를 그대로 따라 하시면 됩니다. 터미널(명령어 창)에 복사-붙여넣기만 하면 돼요.

## 준비물
- Node.js 설치 (https://nodejs.org 에서 "LTS" 버전 다운로드 후 설치)
- Vercel 계정 (https://vercel.com 에서 무료 가입, GitHub/Google 계정으로 가능)

## 1. 터미널 열기
- 맥: Spotlight(⌘+Space)에서 "터미널" 검색
- 이 폴더(vercel-recipe-box)를 다운로드한 위치로 이동:
  ```
  cd 다운로드받은_폴더_경로/vercel-recipe-box
  ```

## 2. Vercel CLI 설치 (최초 1회만)
```
npm install -g vercel
```

## 3. Vercel 로그인
```
vercel login
```
브라우저가 열리면 로그인 진행

## 4. 배포하기
```
vercel
```
질문이 여러 개 나오는데 이렇게 답하면 됩니다:
- Set up and deploy? → y
- Which scope? → 본인 계정 선택
- Link to existing project? → n
- What's your project's name? → **yumigos-recipe** (원하는 이름, 이게 주소가 됩니다!)
- In which directory is your code located? → 그냥 Enter (기본값)
- 나머지 질문들 → 그냥 Enter (기본값)

완료되면 `https://yumigos-recipe.vercel.app` 같은 주소가 나와요. (이름이 이미 사용 중이면 다른 이름을 제안해줘요)

## 5. 데이터베이스 연결하기 (중요! 이거 안 하면 저장이 안 돼요)
1. https://vercel.com 접속 → 방금 만든 프로젝트 클릭
2. 상단 메뉴에서 **Storage** 탭 클릭
3. **Connect Database** 클릭 → **Create New** 탭에서 **Redis** (Upstash) 선택
4. 이름 아무거나 입력하고 리전은 기본값으로 → **Create & Continue**
5. 연결할 환경(Production 등) 선택 → **Connect**

## 6. 다시 배포하기 (데이터베이스 연결 반영)
```
vercel --prod
```

## 7. 완료!
브라우저에서 `https://yumigos-recipe.vercel.app` 접속 → 관리자 비밀번호 설정부터 시작하면 됩니다.
